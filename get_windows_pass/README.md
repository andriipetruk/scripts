
## Installation

```sh
git clone https://github.com/avmaximov/c2_utils.git
cd c2_utils
virtualenv venv
source venv/bin/activate
pip install -r requirements.txt
```

Put edited [c2rc.sh](https://console.cloud.croc.ru/profile/c2rc.sh) to the current directory and

```sh
source c2rc.sh
```
